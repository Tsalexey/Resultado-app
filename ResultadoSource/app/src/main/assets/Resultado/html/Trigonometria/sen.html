Una tabla de senos es una lista de valores de senos de ángulos desde 0° hasta 360°. Mediante el uso de la tabla de senos, usted puede encontrar valores trigonométricos incluso si no tiene una calculadora científica a mano. Para encontrar el valor del seno de un ángulo, simplemente búsquelo en la tabla dada.
