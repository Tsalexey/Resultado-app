Un cilindro es el sólido circunscrito por una superficie cilíndrica y dos planos paralelos (bases) que la intersecan. Un cilindro es circular si su base es una circunferencia. La distancia entre las bases es la altura del cilindro.
Fórmula para calcular el área de la superficie lateral de un cilindro:
S = 2 π R h
Fórmula para calcular el área total de un cilindro
S = 2 π R h + 2 π R 2 = 2 π R(R + h)
donde S es el área, R es el radio del cilindro, h es la altura del cilindro, y π = 3.141592
