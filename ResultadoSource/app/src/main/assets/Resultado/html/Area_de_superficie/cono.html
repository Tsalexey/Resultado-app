Un cono es el sólido resultante de combinar todas las semirrectas que, proviniendo de un punto común (el vértice del cono), intersecan un plano. También se puede definir al cono como el sólido resultante de rotar un triángulo recto alrededor de uno de sus catetos.
Fórmula para calcular el área de la superficie lateral de un cono:
El área de un cuadrado es igual al cuadrado de la longitud de uno de sus lados.

S = π R l

Fórmula para calcular el área de un cono:
S = π R2 + π R l = π R (R + l) donde S es el área, R es el radio del cilindro, h es la altura del cilindro, y π = 3.141592.
