Variación Vnk es la posición de "objetos" en ciertos "lugares" siempre y cuando cada posición quede ocupada por exactamente un objeto y todos los objetos sean diferentes. Más formalmente, una variación de n elementos tomados de k es una selección ordenada de k elementos distintos de un conjunto que tiene nelementos.
Fórmula para calcular el número de variaciones de n elementos tomados de a k
