Calcular

El rango de una matriz es el número de filas (o columnas) linealmente independientes. Utilizando esta definición se puede calcular usando el método de Gauss.

También podemos decir que el rango es: el orden de la mayor submatriz cuadrada no nula. Utilizando esta definición se puede calcular el rango usando determinantes.
