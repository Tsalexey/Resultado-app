La varianza mide la desviación de una variable aleatoria de su valor esperado, de acuerdo con la teoría de probabilidad. Varianza de una variable aleatoria X se escribe como D[X].
Fórmula para calcular la variación de una distribución discreta:

Si X es una variable discreta aleatoria, entonces la varianza es igual a
D[X] = M[X2] - (E[X])2
