Interpretación geométrica
Multiplicar un vector no nulo por un número tiene como resultado un vector colineal con el original (o sea que tiene la misma dirección si el número es positivo, o la dirección opuesta si el número es negativo). El módulo del vector resultante es igual al módulo del vector original multiplicado por el valor absoluto del número.

Interpretación algebraica
Multiplicar un vector no nulo por un número tiene como resultado un vector cuyas coordenadas son iguales a las coordenadas correspondientes del vector original, multiplicadas por el número.
