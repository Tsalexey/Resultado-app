Introduzca los coeficientes de la ecuación bicuadrada:

x 4  +  x 2  +  = 0
Resolver

Una ecuación bicuadrada es una ecuación con la siguiente forma a x4 + b x2 + c = 0, donde a es distinto de 0.
Resolver una ecuación bicuadrada significa encontrar los valores de xique satisfacen la igualdad a xi4 + b xi2 + c = 0.

Si la ecuación bicuadrada contiene signos negativos, coloque un signo de menos (-) delante de los coeficientes necesarios. Por ejemplo:
x4 - 4x2 - 5 = 0, coeficientes 1, -4, -5.
Si la ecuación bicuadrada tiene menos de tres términos, coloque un cero (0) delante de los términos relevantes en la calculadora. Por ejemplo:
x4 - 4x2 = 0, coeficientes 1, -4, 0.
