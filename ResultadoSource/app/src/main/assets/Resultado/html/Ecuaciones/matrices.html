Resolver

Si colocamos los coeficientes de variables desconocidas xi en la matriz A, las variables desconocidas en un vector fila X y los términos libres en un vector columna B, entonces el sistema de ecuaciones lineales se reduce a la siguiente ecuación matricial
A ⋅ X = B,
la cual tiene una única solución solo en el caso de que el determinante de la matriz A sea distinto de 0.

Puede resolver el sistema de ecuaciones de la siguiente forma:
X = A-1 ⋅ B,
donde A-1 es la matriz inversa, que se halla usando la calculadora para determinar matrices inversas o la calculadora para hallar la matriz inversa mediante cofactores.
