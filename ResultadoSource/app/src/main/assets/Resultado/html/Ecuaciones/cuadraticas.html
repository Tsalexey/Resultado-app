Resolver

Una ecuación cuadrática es una ecuación con la siguiente forma a x2 + b x + c = 0, donde a es distinto de 0.

Resolver una ecuación cuadrática significa encontrar los valores de xi que satisfacen la igualdad a xi2 + b xi + c = 0.

Para resolver una ecuación cuadrática, calcule el discriminante del polinomio
D = b2 - 4 a c.
  • Si D > 0, la ecuación tiene dos raíces reales distintas.
  • Si D = 0, la ecuación tiene una única raíz (x1 = x2).
  • Si D ><0, la ecuación no tiene raíces reales.
